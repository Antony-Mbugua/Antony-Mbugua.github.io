Antony's Minimal Mistakes Portfolio
==================================

This repo was bootstrapped to use Minimal Mistakes (remote theme) with a writeups collection organized into categories:
- Ethical-Hacking
- CTF
- SOC-Analysis
- Forensics

IMPORTANT:
- Add real images under assets/images/writeups (example filenames used in front matter).
- Replace /assets/images/avatar.png with your profile image.
- If you want to preview locally, install Ruby, Bundler, then run: undle install and undle exec jekyll serve.
